☁️ Backup automático na nuvem

Desativado
📖 Como configurar (uma única vez)
1. Crie uma conta grátis em supabase.com e um novo projeto.
2. Abra o SQL Editor, cole o código abaixo e clique em Run:
create table if not exists backup (
  id int primary key,
  payload jsonb,
  updated_at timestamptz default now()
);
alter table backup enable row level security;
create policy "acesso_anon" on backup
  for all using (true) with check (true);
3. Vá em Settings → API e copie a Project URL e a chave anon public.
4. Cole os dois campos acima e toque em "Salvar e ativar". Pronto — cada despesa salva será enviada automaticamente.
